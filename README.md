# UMORDA

Universal Multi-Domain Autonomous Decision-Making Agent using Reinforcement Learning.

## Project Goal
This project explores reinforcement learning agents across multiple real-world inspired environments including:
- Hospital resource management
- Traffic signal optimization
- Energy management
- Agriculture
- Finance

The project uses Gymnasium environments, Q-learning, and multi-objective reward optimization.

## Technologies
- Python
- Gymnasium
- NumPy
- Matplotlib
- Streamlit
- Groq API
